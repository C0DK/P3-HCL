**************************************************************
CFG
**************************************************************

Program --> Commands $

Commands --> Command Commands
         | EPSILON

Command --> Declaration
          | Assignment
          | Expression

Type --> number
       | text
       | tuple
       | list
       | dict
       | bool
       | none
       | func
       | var

Expression --> FunctionCall
             | Value
             | LambdaExpression

Value --> Literal
        | identifier

Literal --> literalNumber
          | literalText
          | literalTuple
          | literalList
          | literalDict
          | literalBool

Statements --> Statement linebreak Statements
             | EPSILON

Statement --> VarDcl
            | VarAssign
            | Expression
            | ReturnStatement

Declaration --> Type identifier
              | Type Assignment

Assignment --> identifier equals Expression

LambdaExpression --> curlyL Statements curlyR
                   | LambdaParams curlyL Statements curlyR

LambdaParams --> parenL FunDclParamList parenR arrow Type

FunDclParamList --> FunDclParameter FunDclSecondaryParameter
                  | EPSILON

FunDclParameter --> Type identifier

FunDclSecondaryParameter --> comma FunDclParameter FunDclSecondaryParameter
                           | EPSILON

FunctionCall --> FunCallParameter identifier FunCallParamList

FunCallParamList --> FunCallParameter FunCallParameterList
                   | parenL FunCallParameter FunCallParameterList parenR
                   | EPSILON

FunCallParameter --> Expression
                   | EPSILON

ReturnStatement --> return Expression
